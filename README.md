# lovebirds
For the lovebirds but not by lovebird!

## Requirements

- Node.js

## Install

```cli
npm install -g lovebird
```

## Commands

- [x] `excuse` (Get current user)
- [x] `reaction` (See all posts created by current user)
- [ ] `compliments` (See all posts made by by current user)
